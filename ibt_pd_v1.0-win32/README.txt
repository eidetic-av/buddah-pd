===============

A Pd external of the real-time beat tracker IBT (INESC Porto Beat Tracker), based on MARSYAS

Inlet: Audio signal in the form of sound file or microphone input
Outlet1: Bangs on the beat
Outlet2: Median tempo (BPM) of the last 10 beats

To Install
==========

- Copy the ibt_pd.dll object to your Pd search path
- Copy the help-ibt_pd.pd patch to your Pd search path
